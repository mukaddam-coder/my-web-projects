Vanila JavaScript

Vanilla JavaScript refers to the pure, unadulterated version of JavaScript, free from any libraries or frameworks. It is the core language that powers the web, enabling developers to create dynamic, interactive, and engaging web applications.

React

React is a popular open-source JavaScript library used for building user interfaces, particularly for single-page applications where a smooth and responsive user experience is crucial. Developed and maintained by Facebook, React has gained widespread adoption due to its efficient rendering, reusable components, and robust ecosystem.

Node.js

Node.js is a powerful and versatile open-source JavaScript runtime environment that allows developers to execute JavaScript code outside of a web browser. Built on Chrome's V8 JavaScript engine, Node.js enables the creation of server-side and networking applications with ease.

MongoDB

MongoDB is a leading open-source NoSQL database known for its flexibility, scalability, and performance. Unlike traditional relational databases, MongoDB stores data in a flexible, JSON-like format called BSON (Binary JSON), making it ideal for handling large volumes of unstructured or semi-structured data.
